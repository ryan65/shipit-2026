Tasks
GETGet all tasks
curl http://localhost:3000/api/tasks
POSTAdd a new task
curl -X POST http://localhost:3000/api/tasks \
  -H "Content-Type: application/json" \
  -d '{"name":"My Task","description":"Task description"}'
PUTEdit a task (replace :id with actual task ID)
curl -X PUT http://localhost:3000/api/tasks/:id \
  -H "Content-Type: application/json" \
  -d '{"name":"Updated Name","description":"Updated description"}'
DELETEDelete a task (replace :id with actual task ID)
curl -X DELETE http://localhost:3000/api/tasks/:id
Logs
GETGet all logs as JSON
curl http://localhost:3000/api/logs
GETGet raw log file as plain text
curl http://localhost:3000/api/logs/raw
